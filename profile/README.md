## 📅 스터디 규칙

### 1. 정기 모임
- **시간**: 매주 화요일
- **장소**: 오프라인(역삼)
- **진행 방식**:
  1. **Code Review**: 각자 푼 문제 중 **가장 공유하고 싶은 1문제** 선정 및 발표 (접근 방식, 트러블 슈팅)
  2. **Troubleshooting**: 실패했거나 비효율적인 코드를 함께 보며 해결책 토론

### 2. 문제 풀이 목표
각자의 수준에 맞는 문제를 **스스로 선정**하여 풀이합니다.
- **기본 목표**: 주 **3문제** 이상
- **Class 2 이하**: (브론즈/실버 하위) 위주라면 **4문제** 이상

### 3. 🚨 벌금 제도 (Coffee Rule ☕️)
목표를 달성하지 못하면 팀원들의 카페인을 책임집니다.
- **미달성 문제 1개당 = 커피 1잔**
- **편의를 위하여 미달성 문제 1개당 3천원 적립**
